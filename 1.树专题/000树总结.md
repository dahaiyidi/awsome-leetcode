## 二叉搜索树

- 二叉搜索树是比较特殊的树，特点主要是：
  - 其中序遍历结果是递增的

  - 在二叉搜索树的搜索过程和分治非常类似。因此很多题目会和分治结合起来。最多的时候就是使用1/2的分治
    - 使用递归实现
    
    - ```c++
              int mid = left + ((right - left) >> 1);
              TreeNode* root = new TreeNode(nums[mid]);
              root->left = fun(nums, left, mid - 1);
              root->right = fun(nums, mid + 1, right);
      ```
