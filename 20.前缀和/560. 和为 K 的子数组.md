## Problem

#### [560. 和为 K 的子数组](https://leetcode.cn/problems/subarray-sum-equals-k/)

给你一个整数数组 `nums` 和一个整数 `k` ，请你统计并返回 *该数组中和为 `k` 的子数组的个数* 。

 

**示例 1：**

```
输入：nums = [1,1,1], k = 2
输出：2
```

**示例 2：**

```
输入：nums = [1,2,3], k = 3
输出：2
```

```C++
class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        // 前缀和
        unordered_map<int, int> mp;
        int res = 0;
        mp[0] = 1; // 0就对应着不选
        int pre = 0; // 前缀和
        for(int n : nums)
        {
            pre += n;
            if(mp.find(pre - k) != mp.end())
            {
                res += mp[pre - k];
            }
            mp[pre]++;
        }
        return res;
    }
};
```
